Detailed Description

template<std::derived_from< GridReader > PieceReader>
class GridFormat::VTK::PXMLReaderBase< PieceReader >

Constructors of derived classes for readers of unstructured grids may expose the merge_exceeding_pieces option. If set to true, then parallel I/O with less ranks than pieces in the PVTK file is done such that the last rank reads in and merges all remaining pieces. Otherwise, only as many pieces as ranks are read. On the other hand, if there are more ranks than pieces, some ranks will not read in any data (i.e. the grids are empty).
